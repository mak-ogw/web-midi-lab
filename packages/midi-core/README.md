# @web-midi-lab/midi-core

Shared Web MIDI primitives.

## Scope

- Web MIDI access wrapper
- device listing/snapshots
- raw MIDI send helpers
- MIDI input subscription helpers
- MIDI message formatting/decoding utilities

## Public API

Import from the package root:

```ts
import { createWebMidi, decodeMidiMessage } from '@web-midi-lab/midi-core';
```
